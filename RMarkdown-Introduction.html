Introduction

This presentation has been produced in R Markdown.

R Markdown is part of the RStudio suite and is available in standard RStudio and RStudio server.

It enables users to quickly produce dynamic documents.

Combines the core syntax of markdown with embedded R code chunks so their output can be included in the final document

Back-end Process

Rmd: A combination of markdown and R Chunks

Knitr: The engine which executes the R code within a markdown documents and combines to create overall md

Pandoc: A free and open-source programme which converts documents types. In this case md to X,Y,Z

Types of output and uses

R Markdown supports outputs formats such as HTML, PDF, MS Word & PowerPoint & websites

Manage and disseminate EDA or DS projects with fully reproducible R Markdown documents.

Integrate python scripts and objects within R. Other languages can also be used such as SQL, Shell.

Automate the generation of interactive reports and dashboards with custom template design.

Interact with HTML to create automated email report or create conditional parameterised reports.
